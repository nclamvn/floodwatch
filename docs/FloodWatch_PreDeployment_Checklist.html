üîê 1. Secrets & Environment Configuration
  • Generate production secrets
    Create strong random tokens for ADMIN_TOKEN, SECRET_KEY, POSTGRES_PASSWORD
    ./infra/scripts/generate_secrets.sh
  • Create .env.prod from template
    Copy .env.prod.example ‚Üí .env.prod and fill in all REPLACE_* placeholders
    cp .env.prod.example .env.prod && nano .env.prod
  • Fill in external service credentials
    Mapbox token, Cloudinary (cloud_name, api_key, api_secret), optional Telegram token
  • Validate .env.prod configuration
    Ensure no placeholders remain, DEBUG=false, ALLOW_HTTP=false
    grep -E "REPLACE_|CHANGE_ME_|DEBUG=true|ALLOW_HTTP=true" .env.prod
    Expected: No output (empty)
  • Set secure permissions on .env.prod
    chmod 600 .env.prod && ls -la .env.prod
    Expected: -rw------- (600)
üåê 2. Server & Network Readiness
  • DNS records configured and propagated
    floodwatch.vn and api.floodwatch.vn ‚Üí server IP
    dig floodwatch.vn +short && dig api.floodwatch.vn +short
  • Firewall ports open (22, 80, 443)
    sudo ufw status | grep -E '22|80|443'
  • Docker & Docker Compose v2 installed
    docker --version && docker compose version
  • Repository cloned to /opt/floodwatch
    ls -la /opt/floodwatch/infra/scripts/
  • All deployment scripts executable
    chmod +x /opt/floodwatch/infra/scripts/*.sh
  • SSL certificate ready (or will request via Let's Encrypt)
    If first-time, ensure port 80 accessible for ACME challenge
üè• 3. Capacity & Health Baselines
  • Run preflight checks (9 validations)
    cd /opt/floodwatch && ./infra/scripts/preflight.sh
    Expected: All checks passed ‚úÖ
  • Disk usage < 70% (target: < 80% critical)
    df -h / | awk 'NR==2 {print "Disk: "$5}'
  • Inode usage < 70%
    df -i / | awk 'NR==2 {print "Inode: "$5}'
  • Available memory > 1 GB (minimum 512 MB)
    free -h | grep Mem
  • System time and timezone correct (Asia/Ho_Chi_Minh)
    date && timedatectl | grep "Time zone"
